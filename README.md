# GamePoki Clone - Website Mirip Poki
Website sederhana mirip Poki dengan game unggulan Subway Surfers. Bisa diunggah ke GitHub Pages.

## 🚀 Cara Menggunakan
1. Unduh semua file dalam folder yang sama
2. Buka `index.html` dengan browser untuk melihat hasilnya
3. Buat repositori baru di GitHub
4. Unggah semua file ke repositori tersebut
5. Aktifkan **GitHub Pages** di pengaturan repositori untuk mendapatkan tautan website

## ✨ Fitur
- Tampilan mirip situs Poki
- Game unggulan Subway Surfers
- Tampilan yang bagus di HP maupun komputer
- Kode sederhana mudah diubah

## 📝 Catatan
Kode ini adalah kerangka dasar. Untuk memasukkan game asli, kamu perlu mendapatkan izin dan tautan penyemat dari pemilik game resmi.
